LAN Hardware and Routers

A server provides access to resources to devices connected to it, via a network. This can be a LAN (within say, an office), or a WAN (connected to the internet).

Devices on the network can make requests. Servers filfil them. There are Different types of servers for individual purposes, like hosting a website or managing print requests from users.

Network hardware devices

Any device that connects to a network must have an NIC card, which is a Network Interface Card.

They all contain a unique MAC address which identifies the device, used to tell devices apart within a network.

For wired network cards, they have an ethernet prot.

for Wireless network cards, they have an antenna to transmit / recieve signal via microwave signals.

Devices don't connect directly to each other - they always go thru a router / wireless access point.

Mac addresses and Devices

Mac addresses stand for MEdia Access control.

they are unique to the specific netwrok card unit, associated with a device. It includes information about the manufacturer of the card and the serial number of the device.

They're set by the manfuacturer and cannot be changed, although the user can technically "spoof" their mac address to appear to the network to be a different address, for privacy, to change the address randomly regularly.

a wireless Hub is essentially a splitter for Ethernet ports. They send data that comes from the router out to every single device connected. This reduces performance.

a Switch is similar to a hub, but it can identify the devices connected to it, to forward a packet from the router only to the intended device. Reduces unnecessary transmissions. Full duplex, data can move in both directions via the ethernet cables at the same time.

a Bridge is used to connect two LANs together when they're using the same protocol. a bridge checks the MAC address, and if the device isn't connected to one hub, it forwards it to another hub. PRevents the hub from overcrowding the network by sending packets to EVERY device in the entire network.

a Gateway connects LANs togehter when they're using different protocols. They do conversions of data packets between them.

a Router connects any kind of network using the same protocol - WAN (the internet), LAN (within a building/office).

Routers know which devices are connected to which port, to send the packet back to the correct device using the destination IP address.....if the device is not in its own network, it forwards it to another router using the most efficient route.

Modems are effectively an adaptor for the analog signal transmitted via telephone line infrastructure, and convert it to/from digital form. Can be software.

a repeater is just a device that takes in a weak signal and repeats it and transmits it again. This lets you expand the range of the signal, as signal gets weaker over distance. This allows the same singal from the router to travel further.
